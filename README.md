# CoinWatch — Cross-Chain Crypto Watchlist

A lightweight Chrome extension for tracking tokens on **any chain** — ETH, SOL, Base, BSC, Arbitrum, and 20+ more — in a clean, minimal interface.

Designed for **clarity, speed, and daily use** — not noise.

---

## Overview

CoinWatch is a minimal watchlist that lets you:

- Paste a **contract address, symbol, or pair** from any chain
- Instantly resolve the asset and start tracking
- Monitor price, market cap, liquidity, and 24h change
- Set alerts, favorite tokens, and sort your way

No accounts. No dashboards. No clutter. Just signal.

---

## Features

- **Cross-chain tracking** — Solana, Ethereum, Base, BSC, Arbitrum, Polygon, Avalanche, Optimism, and more
- **Smart input** — paste any contract address, symbol, or Binance pair
- **Price alerts** — set thresholds on market cap, price, or 24h change with quick 2x/5x/10x multipliers
- **Chart site integration** — one-click add from DexScreener, Birdeye, GeckoTerminal, DEXTools, GMGN, Defined
- **Right-click add** — select any contract address on any page to add it
- **Filter by chain** — dynamic chain chips to filter your watchlist by network
- **Keyboard shortcuts** — navigate, search, and manage your list without clicking
- **Import/Export** — backup and restore your watchlist as JSON
- **Light & dark themes**
- **Freemium** — 5 tokens free, up to 30 with Pro ($1.99/mo)

---

## Data Sources

- **DexScreener** — primary source for real-time price, liquidity, and pair data across all chains (batch endpoint)
- **DexPaprika** — fallback for when DexScreener is unavailable (free, no API key, commercial use)
- **Binance** — 24h ticker data for major pairs
- **CoinGecko** — market cap data for Binance pairs (single batched call)
- **CoinCap** — token icons for Binance pairs
- **Jupiter** — Solana token metadata, holder counts, and audit flags

## Installation

1. Download and unzip the extension
2. Open `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked**
5. Select the extension folder

---

## Usage

### Add an asset

Paste any of the following into the input:

- Contract address from any chain (0x... or Solana mint)
- Token symbol (e.g. `PEPE`, `BONK`)
- Binance pair (e.g. `BTCUSDT`, `ETHUSDT`)

### From chart sites

Visit DexScreener, Birdeye, GeckoTerminal, DEXTools, GMGN, or Defined — a floating "Add to CoinWatch" button appears automatically.

### Right-click

Select any contract address on any webpage, right-click, and choose "Add to CoinWatch".
